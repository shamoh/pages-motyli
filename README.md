# Test it locally

```shell
bundle install
bundle exec jekyll serve
```

Using docker (https://hub.docker.com/r/alpine/bundle/):

```shell
docker run -ti -v $(pwd):/apps alpine/bundle bash
```

See http://127.0.0.1:4000.
